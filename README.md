### IDE
主要使用jupyer lab, 並搭配以下extensions:
* Jupyterlab-toc: 可以顯示jupyter內markdown的大綱
* jupyterlab-go-to-definition
* nbdime: 比較不同commit的ipynb的差異
* jupyterlab-variableInspector: 在jupyter lab內觀察變數的內容
